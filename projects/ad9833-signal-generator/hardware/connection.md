# AD9833 信号源 — 硬件连接

## 使用的芯片与模块

| 器件 | 型号 | 说明 |
|------|------|------|
| MCU | STM32F103C8T6 (Blue Pill) | Cortex-M3, 72MHz |
| DDS | AD9833 模块 | 可编程波形发生器, 最高12.5MHz |
| 晶振 | 25MHz (模块自带) | AD9833主时钟 |

## 引脚连接表

| STM32F103C8T6 | AD9833 模块 | 功能 |
|---------------|-------------|------|
| PA5 (SPI1_SCK) | SCLK | SPI 时钟 |
| PA7 (SPI1_MOSI) | SDATA | SPI 数据输出 |
| PA4 (软件GPIO) | FSYNC | SPI 片选/帧同步 |
| 3.3V | VCC | 电源 (3.3V) |
| GND | GND | 地 |
| — | AGND | 模拟地 (与GND共地) |
| — | VOUT | 信号输出 (接示波器/后级) |

## 连接示意图

```
STM32F103C8T6              AD9833 Module
┌─────────────┐           ┌──────────────┐
│         PA4 │──────────▶│ FSYNC        │
│     SPI1_SCK│           │              │
│         PA5 │──────────▶│ SCLK         │
│    SPI1_MOSI│           │              │
│         PA7 │──────────▶│ SDATA        │
│             │           │              │
│       3.3V  │──────────▶│ VCC          │
│        GND  │──────────▶│ GND / AGND   │
│             │           │              │
│             │           │ VOUT ──▶ 示波器
└─────────────┘           └──────────────┘
```

## 注意

1. **共地**：AD9833 模块的 AGND 和 DGND 都要连到 STM32 的 GND
2. **电源去耦**：VCC 引脚附近加 100nF + 10μF 电容滤波
3. **SPI 模式**：AD9833 使用 SPI Mode 2 (CPOL=1, CPHA=0)，数据在下降沿写入
4. **FSYNC**：传输前拉低，传输完成后拉高，一次传输 16 bits
5. **输出幅度**：VOUT 输出约 0.6Vpp（峰峰值），如需更大信号需后级放大
6. **频率上限**：理论 MCLK/2 ≈ 12.5MHz，实际稳定范围建议 < 5MHz

## 可选扩展

如果需要更大的输出幅度，在 VOUT 后加一级运放放大：

```
AD9833 VOUT ──▶ [隔直电容 10μF] ──▶ [运放同相放大 5-10倍] ──▶ 输出
                                      e.g. NE5532, OPA2134
                                      双电源 ±5V 供电
```
